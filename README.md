# LPM: Latent Persona Modeling

LPM is a research subproject for modeling character/persona dynamics from
GalGame narrative data. Its first objective is theoretical: formulate the
dataset as a viewpoint-biased observation stream rather than a flat dialogue
corpus, then derive learning problems that can support latent structure
instead of surface style imitation.
